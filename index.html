Puzzle - A Minecraft Server Manager

A simple server manager for the popular game, Minecraft. Built for the Linux terminal, it allows you to start, stop, backup, restore, and set automated backups for your Minecraft server. Given a number of maximum backups, it will auto-remove the oldest backups to make sure you never exceed the maximum amount. This saves tons of storage space each month! Everything is backed onto an SVN repository.

Fortnite Analytics

https://fortniteanalytics.web.app/

A simple web app which displays Fortnite player statistics. It is built with React and makes RESTful API HTTP requests to retrieve player data. The RESTful API is implemented in Java with Spring Boot Microservices, where Fortnite player information is received from fortniteapi.io and organzied appropriately.

Minesweeper - Game & Engine

A simple Minesweeper engine which can solve Easy, Intermediate, and Expert with success rates of 80%, 50%, and 3% respectively. The engine looks for patterns to open up the board. If no patterns are found, it will make a list of guaranteed mines and select a random unopened square outside of that list.

Learn more about my Minesweeper Engine on my Github here, where you can run an executable to play Minesweeper yourself.

Text-based Message Server Board

A text-based server that stores a 76-byte message. Written for the Linux terminal, it allows multiple clients to access and modify this message by connecting to a port on a Linux machine. It features simple network programming and uses the select system call (see the Linux Man Page for more information) to perform multiplexing. This program was written in C.
