Puzzle - A Minecraft Server Manager

Built for the Linux terminal, it allows you to start, stop, backup, restore, and set automated backups for your Minecraft server, where everything is backed onto an SVN repository. It also features a web option, where you can control the server remotely. The web app is built with React, and implements RESTful API HTTP requests with Spring Boot in Java, to run shell scripts on the Linux server (visit Puzzle Web).

Learn more about Puzzle on my GitHub here.

Scrabble Calc

https://scrabblecalc.web.app/

A simple web app tool to help you automate score calculations during your Scrabble games. It can also store all your Scrabble games for your to view later and display high scores like most points in a single game and most points from a single word. It uses Node.js to implement a GraphQL API that stores and retrieves Scrabble data from a MongoDB. The frontend is written with React where it uses ApolloClient to make queries and mutations with the GraphQL API.

Learn more about Scrabble Calc on my GitHub: Frontend Repo and GraphQL API Repo.

Minesweeper - Game & Engine

A simple Minesweeper engine which can solve Easy, Intermediate, and Expert with success rates of 80%, 50%, and 3% respectively. The engine looks for patterns to open up the board. If no patterns are found, it will make a list of guaranteed mines and select a random unopened square outside of that list.

Learn more about my Minesweeper Engine on my Github here, where you can run an executable to play Minesweeper yourself.
